# HTML Selector for 1C

Інструмент для візуального вибору елементів HTML-сторінок та генерації коду 1С (BSL) з шаблонів.

## Встановлення та запуск

```bash
npm install
npm run dev
```

Відкрий http://localhost:5173/

## Короткий опис

Веб-застосунок дозволяє:
- Завантажити HTML-файл як шаблон
- Візуально виділяти елементи на сторінці
- Налаштовувати CSS-селектори з параметрами (contains, exact, starts, ends)
- Будувати ієрархію селекторів (вкладені елементи)
- Генерувати код 1С для підстановки даних у шаблон

## Як користуватись

### 1. Завантаження шаблону

1. Натисни **Choose HTML File**
2. Вибери `.html` файл з диска
3. Шаблон відобразиться в iframe, ім'я файлу збережеться в базі

### 2. Вибір елементів

1. Обери тип елемента у випадаючому списку (All Types, div, table, td тощо)
2. Натисни **Select Elements**
3. Клікай на потрібні елементи в iframe — вони підсвічуватимуться
4. Натисни **Done Selecting** коли закінчив

### 3. Редагування селекторів

Для кожного селектора в списку можна налаштувати:

| Поле | Опис |
|------|------|
| type | Тип даних (напр. `string`) |
| key | Ім'я змінної для підстановки в 1С |
| + param | Додати фільтр за атрибутом |
| + sub | Додати вкладений елемент |
| × | Видалити |

#### Параметри (params)

Фільтри для уточнення селектора:
- **key** — ім'я атрибута (class, id, data-* тощо)
- **type** — спосіб збігу: any, contains, exact, starts, ends
- **value** — значення для порівняння

### 4. Підсвічування

- **Show Elements** — показати всі виділені елементи
- **Hide Elements** — сховати підсвічування

### 5. Збереження та імпорт

- **Export JSON** — зберегти всі селектори у JSON-файл
- **Import Selectors** — відкрити вікно вибору іншого шаблону та скопіювати з нього селектори (тільки ті, що знайдені в поточному HTML)

### 6. Генерація коду 1С

1. Перейди на вкладку **Result**
2. Натисни **Generate**
3. Код з'явиться в текстовому полі
4. Натисни **Копіювати** для копіювання в буфер

#### Опції генерації

- **create function** — обгорнути результат у функцію 1С
- **function name** — назва функції (за замовчуванням `GetHTMLTemplate`)

#### Результат

Код 1С (BSL) типу:

```1c
Function GetHTMLTemplate(Val Data) Export
    tempData = New Structure("name, title");
    FillPropertyValues(tempData, Data);

    HTMLText = "";
    HTMLText = HTMLText + "<html>...";
    HTMLText = HTMLText + "<div>" + XMLString(tempData.name) + "</div>";
    ...
    Return HTMLText;
EndFunction
```

## Структура файлів

```
src/
  main.ts          — завантаження HTML, iframe, вибір елементів
  selectorEdit.ts  — редагування селекторів та суб-селекторів
  generateCode1c.ts — генерація коду 1С
  style.css         — стилі (light/dark)
  db/               — SQLite база (автоматично)
vite.config.ts     — Vite + SQLite API middleware
```

## База даних

SQLite зберігається в `src/db/database.sqlite`. Таблиці:
- `templates` — HTML-шаблони
- `selectors` — селектори з прив'язкою до шаблону
- `parameters` — фільтри селекторів
